Advance Data Structures

We learnt about strings, integers, float, boolean etc. in the last resource. Now, it's time to expand that knowledge to talk about data structures, i.e. ways in which we can organize these in logical groups that can then be used together. For instance, what if we wanted to process a list of students and their marks to generate a report that contains the names of students who have scored more than 40 marks out of 60? One way to do that is to create multiple variables containing each of the student names, and then yet other variables containing their scores, and then using "if" condition to check if the score obtained by a student is greater than 40 and ultimately printing that student name. In other words, the code will look something like this.

student1Name = "Kartik"

student2Name = "Karan"

student1Marks = 60

student2Marks = 20

if student1Marks > 40:

print(student1Name)

if student2Marks > 40:

print(student2Name)

Here, the program will correctly print "Kartik" because Kartik's marks>40. But, imagine that we wanted to do this with 500 students. We will then be working with 1000 variables which is crazy. Here is where data structures come in.

List

The first data structure we will discuss is a list. As the name suggests, a list is simply a list of values. For instance, in the last example, instead of creating a variable for every student, we can create a list of all of the student names as follows.

studentNames = ["Kartik", "Karan"]

Note the [ and ]. This is used to declare a list and comma-separated string values are the elements of the list. Here are a few other examples of lists.

numbers = [1, 4, 6, 10]

floatVals = [2.5, 5.3, 3.3, 4.6]

listOfMultipleDatatypes = [2.5, 2, "test"] # This is valid. You can have a list consisting of multiple data types, although this doesn't make sense in most cases.

Just like the "for" loop we discussed previously where the counter starts from 0, lists are zero-indexed as well (in general, everything in computer science is). This means that the first element of a list is actually represented as the 0th element, the second element as the 1st element and so on. Given this information, we use the listName[indexOfElement] to get the value at a particular index within the list. For instance, if I wanted to print the first element in the numbers list defined above, here is how I would do it.

print(numbers[0])

numbers[0] means the element at the 0th index, i.e. the first element. The index goes upto the length of the list-1. If you try to get a value beyond this, the interpreter will throw an "out of bounds" exception. Interestingly, the index can also be negative. for instance, I can write the following:

print(numbers[-1])

If the index is negative, then the counting starts from the end of the list. So, -1 indicates the last element, -2 indicates the second last element and so on. The print statement above will thus print the last element in the list.

A few other functions that are important to remember include the following:

  • list.append(elementToBeAppended) function is used to append or add the value at the end of the list. For instance, numbers.append(4) will add the value 4 at the end of the list.
  • list.insert(position, elementToBeInserted) inserts the element specified as the second argument at the position indicated by the first argument. e.g. numbers.insert(2, 9) inserts 9 at index 2.
  • list.remove(elementToBeRemoved) will remove the element passed to the function from the list if it exists in the list, otherwise will throw an error. For instance, numbers.remove(4) will remove the value 4 we recently appended to the list.
  • list1.extend(list2) merges the contents of list1 and list2 into a single list. e.g. Consider the following.

list1 = ["Kartik", "Karan"]

list2 = ["Mehak", "Shradha"]

list1.extend(list2)

print(list1)

The output will be the elements of list2 merged into list1, i.e. Kartik, Karan, Mehak, Shradha.

  • len(list) returns the length of the list. e.g. len(numbers) will return the No. of elements in the list.
  • reversed(list) reverses the list. reversed(numbers), for example, will reverse the elements of the list such that the last element becomes the first and so on.

Please note that string is also essentially a list of characters. Consider the following example.

str = "Hello"

print(len(str)) #the output will be 5

print(str[0]) #the output will be H

Iterating a List

Using our understanding of for loop, we an iterate a list as follows.

for i in range(len(list)):

print(list[i])

Here, we are going from 0 to len(list) not included and printing each of the values, thereby printing the list. Python provides another way, however, to iterate a list.

for elem in list: # elem is just a variable name, you can use any other variable name

print(elem)

This will directly print the various elements in the list. You don't have to worry about range or index here.

Dictionaries

In the previous example of students and their marks, there is an association between the student and their marks. We previously modeled this using a list of students and a list of marks with the elements of both the lists mapping to each other. But that is not the most efficient way to represent this structure. A better way is to use what is known as a dictionary. A dictionary consists of a key and an associated value. In this case, for example, the key will be the name of the student and the value will be their marks. This helps us encapsulate all information about a student and their marks in one data structure. Here is how it works.

studentDictionary = {} # This is used to initialize a dictionary

studentDictionary["Kartik"] = 60

studentDictionary["Karan"] = 20

Here, the dictionary has two keys "Kartik" and "Karan" each containing a value 60 and 20 respectively. Once added to the dictionary, the value for a particular key can be obtained as follows:

valueForKey = nameOfTheDictionary[key]

In other words:

marksForKartik = studentDictionary["Kartik"] # Where Kartik is the key

Further, you can get a list of all of the keys and values in the dictionary using the dictionary.keys() and dictionary.values() function. e.g.

listOfKeys = studentDictionary.keys() #This will give me a list of the keys in the studentDictionary

listOfValues = studentDictionary.values() #This will give me a list of the values in the studentDictionary

Finally, note that the value can be anything-an integer, a string, a list or even another dictionary.

Sets

Just like a set in mathematics, a set in computer science is essentially a list but without any duplication, i.e. all elements in a set are unique elements. A set is declared as follows.

a = set([list of values])

e.g.

a = set([4, 4, 5, 5, 6]) #This will be stored as set([4, 5, 6]) since all dupicate values will be removed

All other operations and traversal methods remain the same as a list.

Tuples

Tuples are used to store multiple items in a single variable. This is a collection which has a specific order to it and is unchangeable. Consider the following example.

tupleOfStudents = ("Kartik", "Karan")

Just like lists, tuples are zero-indexed and can have various data types within the same tuple. The individual elements of a tuple can also be accessed in a similar way as a list as follows.

print(tupleOfStuents[0]) #Will print the first element of the tuple, i.e. Kartik

Tuple as a data structure is relatively less used, but it's good to have it in our toolbox.
